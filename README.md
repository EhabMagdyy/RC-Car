# Features 
    - remote controlled car.
    - obstacle avoidance.
    - changing speed is possible.

# Tools
    - STM32F401RCT6 Microcontroller.
    - ESP32 Microcontroller.
    - Ultrasonic sensor HC-SR04
    - L298N motor driver.
    - ×4 DC Motor.
    - buzzer.
    - ×3 18650 battery 3.7v.
